设计模式 (4) - 适配器模式

Posted by Littlesqx on January 8, 2018

好颓好颓~

简单介绍

适配器模式(Adapter Pattern) ：是作为两个不兼容的接口之间的桥梁，将一个接口转换成客户希望的另一个接口，适配器模式使接口不兼容的那些类可以一起工作，其别名为包装器(Wrapper)。

PHP 实现

假设旧系统都是采用 MySQLi 的方式连接数据库，当项目移植到另一个系统，新环境不支持 MySQLi ，而支持 PDO ，适配器的作用就体现了；反之也是类似的。

这是旧代码：

<?php

$config = [
	'host' => 'localhost',
	'dbname' => 'test',
	'username' => 'homestead',
	'password' => 'secret'
];

interface MySQLi {
	public static function connection($config);
}

class Db implements MySQLi
{
	public static function connection($config)
	{
		return new mysqli(
			$config['host'],
			$config['username'],
			$config['password'],
			$config['dbname']
		);
	}
}

$db = Db::connection($config);

这是新代码：

<?php

$config = [
	'host' => 'localhost',
	'dbname' => 'test',
	'username' => 'homestead',
	'password' => 'secret'
];

interface IMySQLi {
	public static function connection($config);
}

interface IPDO {
	public static function connection($config);
}

class Db implements IMySQLi
{
	public static function connection($config)
	{
		return new mysqli(
			$config['host'],
			$config['username'],
			$config['password'],
			$config['dbname']
		);
	}
}

class PdoDb implements IPDO
{
	public static function connection($config)
	{
		return new PDO('host='.$config['host'].';dbname='.$config['dbname'], $config['username'], $config['password']);
	}
}

class DbAdapter implements IMySQLi
{
	public static function connection($config)
	{
		$driver = $config['driver'] ?? 'mysqli';

		if ($driver === 'pdo') {
			return PdoDb::connection($config);
		}

		return Db::connection($config);
	}
}

class Client implements IMySQLi
{

	public static function connection($config)
	{
		return DbAdapter::connection($config);
	}
}

$db = Client::connection($config);

补充

其他场景，例如 Cache 驱动的适配（文件、 Redis、memcache等）。
